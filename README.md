##  grade-generator.py
### **What it does**
- Asks the user for grades in different categories  
- Applies the correct weights  
- Calculates:
  - Formative Total (60%)
  - Summative Total (40%)
  - Final Grade
  - GPA
  - Pass/Fail Status
